# shell内置变量有哪些

| 变量 | 含义                                  |
| ---- | ------------------------------------- |
| $0   | 脚本名字                              |
| $1   | 位置参数1                             |
| $#   | 位置参数的个数                        |
| $*   | 所有的位置参数                        |
| $?   | 返回值                                |
| $$   | 脚本的进程ID(PID)                     |
| $!   | 运行在后台的最后一个作业的进程ID(PID) |
